How do we use tags

No Name Attributes Explanation
1 html html has not attribute The element defines the whole document.
2 head head has not attribute The 'head' element is a container for metadata and is placed between the 'html' tag and the 'body' tag.
3 body body has not attribute The 'body' element defines the document body.
4 h1 h1 has not attribute The 'h1' element defines a heading.
5 p p has not attribute The 'p' element defines a paragraph.
There are more than 200 tags in HTML
